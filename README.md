# オンラインクリニック

スマホで相談できるオンライン診療サービス向けの静的ホームページです。

## ローカル確認

```powershell
node serve.js
```

ブラウザで `http://127.0.0.1:4173` を開いて確認できます。

## GitHub Pages

`index.html`、`styles.css`、`script.js`、`assets/` をそのまま公開できます。

## ページ構成

- `/` トップページ
- `/pricing/` 料金一覧
- `/flow/` ご利用の流れ
- `/diet/` ダイエット・サポート
- `/stomach/` 整腸・胃薬
- `/pill/` 低容量ピル
- `/ed/` ED薬
- `/aga/` AGA
- `/beauty/` 保湿剤・美容薬
- `/contact/` お問い合わせ
- `/limited-info/` 限定情報
- `/privacy/` プライバシーポリシー
- `/profile/` 特定商取引法に基づく表記
- `/terms/` 利用規約
